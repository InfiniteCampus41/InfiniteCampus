This Is Infinite Campus
A Hopefully Unblocked Website That Allows You To Play Games, Browse The Web, And Much More.

WARNING:
If You Fork This Repo And Try To Run It, The Proxy Will Not Work
This Is Because This Repo Is Only The Public Folder For The Infinite Campus Server And Since The Proxy Tries To Contact The Server, They Will Fail If The Files Are Not There.
If You Would Like A Working Proxy, Please Check Out The InfiniteCampus-Core Repo

Requirements: Node.js & npm

Step 1: Click The Green Code Icon And Select Download Zip

Step 2: Extract The Zip File And Open The Folder With Everything In It In The Terminal

Step 3: Run npm install

Step 4: Run node server.js

Step 5: Enter The Port You Would Like To Use Ex: 2000

Step 6: Select Either Local Or Network Modes (Network Allows Anyone On Your Network With The Url To Visit It)

Step 7: Go To The Url It Gives You
