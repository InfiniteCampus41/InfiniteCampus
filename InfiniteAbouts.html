Roles info


Infinite Campus has roles. These roles are only appliccable if you are logged in.


To log in, click here

These roles are ranked from most powerful to least

Permissions from lower power roles also carry over to higher power roles

Ex: If Premium T1 can have 500MB file uploads, every role above that also gets it.

The only permissions that do not carry are:

  • Custom subdomain - Premium T3
  • Manage website server - Developer
  • Access to partner chat - partner




Role #1: Owner

This role is for the owner of the site
Permissions:

  • Same as Tester


Role #2 Tester

This role is for test accounts
Permissions:

  • Read and write to everything


Role #3 Co-owner

This role is for people that are very trusted by the owner to moderate
Permissions:

  • Create channels
  • Delete channels
  • Rename channels
  • Verify users
  • Manage all Typing Data
  • Manage server
  • Accept movie Applicants
  • Send emails to subscribed users


Role #4 Head admin

This role is for people that are trusted to moderate
Permissions:

  • Same as admin


Role #5 Admin

This role is for people that are trusted by the owner, but are not active on the site
Permissions:

  • Delete messages sent by roles below
  • Mute roles below


Role #6 Developer

This role is given to people that are coding Infinite Campus
Permissions:

  • Read user emails
  • Add website updates
  • Add future updates
  • Manage website server


Role #7 Premium T3

This role is given to users after donating $10 or more in one donation to the site
Permissions:

  • Access admin chat
  • Make your own subdomain of infinitecampus.xyz OR instructure.space

    • Example: name.infinitecampus.xyz
    To get your subdomain, you must contact the owner


Role #8 Premium T2

This role is given to users after donating $5 - $9.99 in one donation to the website
Permissions:

  • access premium chat


Role #9 Premium T1

This role is given to users after donating $2 - $4.99 in one donation to the website
Permissions:

  • 500MB file uploads in 10 min file upload app
  • Files last 15 mins instead of 10 in 10 min upload app


Role #10 100th user

This role will be given to the 100th user that signs up for the website
Permissions:

  • Same as regular user


Role #11 partnered user

This role is given to users that partner with Infinite Campus
Permissions:

  • Edit own partnership
  • Access to partner chat
  • Same as regular user


Role #12 verified user

This role is given to you once someone verifies you
Permissions:

  • See unrestricted channels
  • Send messages in unrestricted channels
  • Delete own messages
  • Edit own messages
  • Create group chats
  • Create private chats
  • Mention users
  • Change own typing data


Role #13 unverified user

When you first sign up for the website, you will be an unverified user.
To verify, you just need to wait. It usually takes at most 12H to verify
Permissions:

  • Read own private chats
  • Read own mentions
  • Change/Set own pic
  • Change/Set own display name
  • Change/Set own name color
  • Change/Set own password
  • Change/Set own bio


Role #14 spam user

This role is given to people that have engaged in suspicious activity on the website
If you see someone with this role, please do not interact with them
Permissions:

  • Same as unverified user unless verified
  • Cannot access admin tools


Role #15 signed out user

This role is what you have if you are viewing the site not signed in
Permissions:

  • Read user bios
  • Read user pics
  • Read user name colors
  • Read user display names
  • Read user roles
  • Read website updates
  • Read website future updates
  • Send messages as anonymous users
  • Read unrestricted channels


Role #16 Discord user

This role is given to you if you enter in your Discord username in the settings page
This role could also mean that a message in the website chat is from the Discord. In these messages, the outline around the profile picture is always blue.
Permissions:

  • Same as regular user


Role #17 donater

This role is given to you if you donate less than $2 to the website
Permissions:

  • Same as regular user


Role #17 guesser

This role is given to you if you complete a secret task somewhere involving the site.
This role is no longer available
Permissions:

  • Same as regular user


Role #18 muted user

This role is given to you if you have violated the rules or TOS or are just being annoying.
Permissions:

  • Cannot send messages
  • Cannot write to typing
  • Cannot create private chats
  • Cannot send messages in private chats


Role #19 link sharer

This role is given to you if you share a lot of links in the links channel
Permissions:

  • Same as regular user


Role #20 Blocksi user

You can get this role by enabling Blocksi on your account in the settings page
Permissions:

  • Same as regular user


Role #21 GoGuardian user

You can get this role by enabling GoGuardian on your account in the settings page
Permissions:

  • Same as regular user


Role #22 Lanschool user

You can get this role by enabling Lanschool on your account in the settings page
Permissions:

  • Same as regular user


Role #23 Linewize user

You can get this role by enabling Linewize on your account in the settings page
Permissions:

  • Same as regular user


Role #24 Securely user

You can get this role by enabling Securely on your account in the settings page
Permissions:

  • Same as regular user


Role #25 Uploader

You can get this role by uploading a movie, and it getting accepted by an admin
Permissions:

  • Same as regular user
